# Challenge 003 - Data Structures

## Description

Third challenge of the Java Roadmap.
The objective is to practice the main data structures provided by the Java Collections Framework and their basic operations.

## Concepts practiced

- `List` (`ArrayList`)
- `Set` (`TreeSet`)
- `Queue` (`LinkedList`)
- `Map` (`TreeMap`)
- Insertion, update, removal and search operations
- Sorting collections
- User input with `Scanner`
- Exception handling
- Regular expressions for input validation

## Extra challenge

Implemented a terminal-based contact book supporting:

- Search contacts
- Add contacts
- Update contacts
- Delete contacts
- Phone number validation

## Technologies

- Java 26
