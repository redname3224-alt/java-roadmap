## Description

Fifth challenge of the Java Roadmap.
The objective is to understand how Java handles value assignment, reference assignment, and parameter passing for primitive types and objects.

## Concepts practiced

- Primitive types
- Object references
- Value assignment
- Reference assignment
- Passing primitive values to methods
- Passing object references to methods
- Modifying an object's state through a method
- Reassigning object references inside methods
- Returning arrays from methods
- Nested classes

## Extra challenge

Implemented two methods that swap values and return the swapped result:

- Swap two primitive values and return them in an array.
- Swap two object references and return them in an array.
- Verify that the original variables remain unchanged after the method call.

- Java 26
