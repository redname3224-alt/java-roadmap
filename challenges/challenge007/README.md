# Challenge 007 - Stacks and Queues

## Description

Seventh challenge of the Java Roadmap.
The objective is to understand how stacks (LIFO) and queues (FIFO) work by implementing their basic operations using a `List`.

## Concepts practiced

- `List` (`ArrayList`)
- Stack (LIFO)
- Queue (FIFO)
- `push`
- `pop`
- `peek`
- `enqueue`
- `dequeue`
- User input with `Scanner`
- Interactive console applications

## Extra challenge

Implemented two simulations using the custom stack and queue implementations:

- A web browser navigation system with back and forward history.
- A shared printer queue that prints documents in FIFO order.

## Technologies

- Java 26

## Possible improvements

- Replace the `ArrayList` implementations with `Deque` (`ArrayDeque`), which is the recommended structure for stacks and queues in Java.
