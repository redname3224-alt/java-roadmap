# Challenge 000 - Java Syntax, Variables and Data Types

## Description

First challenge of the Java Roadmap.
The objective is to practice basic Java syntax, comments, variables and primitive data types.

## Concepts practiced

- Single-line and multiline comments
- Javadoc comments
- Variables and constants
- Primitive data types
- Console output

## Technologies

- Java 26
