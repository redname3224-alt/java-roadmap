# Challenge 004 - Strings

## Description

Fourth challenge of the Java Roadmap.
The objective is to practice the most common operations that can be performed with strings in Java and implement algorithms to analyze words.

## Concepts practiced

- Character access (`charAt`)
- Substrings (`substring`)
- String concatenation (`concat`, `String.join`)
- String formatting (`String.format`)
- String repetition (`repeat`)
- String splitting (`split`)
- String trimming (`trim`)
- Case conversion (`toLowerCase`, `toUpperCase`)
- String comparison (`equalsIgnoreCase`)
- Search and verification (`contains`, `startsWith`, `endsWith`, `indexOf`, `isEmpty`)
- String replacement (`replace`)
- String traversal with `toCharArray`
- `StringBuilder`
- Arrays and sorting (`Arrays.sort`)
- `HashSet`
- User input with `Scanner`

## Extra challenge

Implemented a word analyzer that checks whether two words are:

- Palindromes
- Anagrams
- Isograms

## Technologies

- Java 26
