# Challenge 002 - Functions and Methods

## Description

This challenge focuses on creating methods with different signatures in Java:

- Methods with no parameters and no return value.
- Methods with parameters.
- Methods with return values.
- Methods with multiple parameters.

It also demonstrates:

- The use of built-in Java methods.
- The difference between local and global variables.
- That Java does not support nested methods.

## Extra Challenge

Implemented a customized FizzBuzz-like method that:

- Prints numbers from 1 to 100.
- Replaces multiples of 3 with the first string.
- Replaces multiples of 5 with the second string.
- Replaces multiples of both 3 and 5 with the concatenation of both strings.
- Returns the number of times a number was printed instead of text.

## Concepts Practiced

- Method declaration and invocation
- Parameters and return values
- Local vs. class variables
- `static` methods
- Loops and conditional statements
- Java Standard Library (`LocalDateTime`, `Scanner`, `String` methods)

## Technologies

- Java 26
