# Challenge 006 - Recursion

## Description

Sixth challenge of the Java Roadmap.
The objective is to understand recursion by implementing recursive methods and applying them to common mathematical problems.

## Concepts practiced

- Recursive methods
- Base cases
- Recursive calls
- Call stack behavior
- Recursive countdown
- Factorial calculation
- Fibonacci sequence
- User input with `Scanner`
- Exception handling
- Input validation

## Extra challenge

Implemented recursive methods to:

- Print numbers from 100 to 0
- Calculate the factorial of a number
- Calculate the value at a given position in the Fibonacci sequence

## Technologies

- Java 26
