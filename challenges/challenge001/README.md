# Challenge 001 - Operators and Control Structures

## Description

Second challenge of the Java Roadmap.
The objective is to practice Java operators and the main control flow structures.

## Concepts practiced

- Arithmetic operators
- Unary operators
- Assignment operators
- Relational operators
- Logical operators
- Bitwise operators
- Conditional (ternary) operator
- Type operator (`instanceof`)
- `if` / `else`
- `switch`
- `for`
- Enhanced `for` (for-each)
- `do-while`

## Extra challenge

Print all even numbers between 10 and 55 (inclusive), excluding 16 and multiples of 3.

## Technologies

- Java 26
